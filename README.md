## Last Saint

## Summary
Last Saint is een game waarbij er allemaal enemies rondom de held verschijnen. Ben jij de held die het langs kan overleven? Probeer het zelf en maak je strategie.

## Genre

2D Shooter
Strategy

## Concept Pitch
Shooter game waarbij de speler zolang mogelijk moet overleven en moet zorgen dat alle vijanden rondom hem dood zijn.

## Gameplay Mechanics

### Game Flow
Speler loopt rond, enemies spawnen rondom hem en hij moet die ontwijken en proberen te vermoorden voor hij overwelmt wordt

### Game Feel

### Gameplay
zolang mogelijk overleven

### Mechanics
Vijanden spawnen rondom de spelern, schieten op vijanden
